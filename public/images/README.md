# Avatar Image Instructions

Please add your Studio Ghibli avatar image to this directory with the filename:
`studio-ghibli-avatar.png`

The image should be:
- High resolution (at least 350x350 pixels)
- PNG format for best quality
- Named exactly: `studio-ghibli-avatar.png`

Once you add the image file, your avatar will automatically appear in the hero section of your portfolio website!
